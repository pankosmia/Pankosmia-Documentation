  1. Environment Requirements

Exigences de l’environnement

Exigences de l’environnement de développement local pour le développement d’un dépôt d’application de bureau basé sur Pankosmia

Table des matières

  • Testé sur :
    • Ubuntu 24.04 avec :
    • Windows 11 avec :
    • MacOS avec :

Testé sur :

Ubuntu 24.04 avec :

  • npm 10.7.0
  • node 20.18.1
  • rustc 1.88.0 – curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Windows 11 avec :

  • git 2.44.0; Voir https://git-scm.com/install/windows
  • npm 10.7.0
  • node 20.18.1
  • rustc 1.88.0 – Voir https://www.rust-lang.org/tools/install
  • cmake 3.31.0 – La version 3 est requise. Voir https://cmake.org/download/
  • Selon votre système, powershell pourrait avoir besoin d’une autorisation pour exécuter des scripts.
  • L’exécution locale de builds exe (optionnel) nécessite l’installation de Inno Setup - testé avec la v6.4.3

MacOS avec :

  • npm 10.7.0 (testé sur Monterey)
  • npm 10.8.2 (testé sur Sequoia)
  • node 20.18.1
  • rustc 1.88.0 – curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • OpenSSL 3.5.0 – brew install openssl
